## 🎯 Overview

This project implements a Course Management System for **Inception BD**, a fictional educational platform. It demonstrates the four pillars of Object-Oriented Programming:

### OOP Concepts Demonstrated

| Concept | Implementation |
|---------|---------------|
| **Classes & Objects** | `User`, `Student`, `Instructor`, `Course` classes with instantiated objects |
| **Inheritance** | `Student` and `Instructor` inherit from `User` base class |
| **Encapsulation** | Private `__enrolled_courses` in Student, protected `_email` in User |
| **Polymorphism** | `display_profile()` overridden in each subclass with unique behavior |
